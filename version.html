How to install or update your Query Generator program

Developed by Josh Swan



  1. Click the link below to download the latest version:

    2. Download QueryGenerator.zip

  2. This will download a ZIP folder called "QueryGenerator" to your downloads folder.
  3. Right-click and select Extract All...
  4. Extract to a convenient location e.g. Desktop
  5. If you already have a version installed in your location, you will be prompted to take action. Select ‘Replace the files in the destination’. This shouldn’t take more than 1 minute. If this doesn’t happen, you may need to manually delete the old QueryGenerator folder to avoid confusion between versions.
  6. If you want to create a shortcut, head into the QueryGenerator folder where you extracted it. Find the QueryGenerator.exe file and right-click (make sure you right-click hovering over its name to bring up more options), and click Send to… Desktop (create shortcut)
  7. You should now be able to open the Query Generator app from your desktop
